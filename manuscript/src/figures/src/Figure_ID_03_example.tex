\caption{\textbf{
User satisfaction ratings for SciTex features
}
\smallskip
\\
Heat map showing user satisfaction ratings for various SciTex features based on a 5-point Likert scale (1 = very dissatisfied, 5 = very satisfied). Data collected from 75 active users over a 6-month period. The highest satisfaction ratings were observed for the automated figure processing (4.8/5.0) and bibliography management (4.7/5.0) features, while the revision tracking feature received the lowest satisfaction rating (3.9/5.0).
}
